# Carpeta de Documentos

Esta carpeta contiene los documentos que el bot envía a los usuarios.

## Archivo requerido:

- **formulario_renuncia.pdf** - Formulario de renuncia de derechos de agua

### Instrucciones:

1. Guarda el formulario con el nombre exacto: `formulario_renuncia.pdf`
2. El bot detectará automáticamente cuando el documento esté disponible
3. Para actualizar el formulario, simplemente reemplaza el archivo con el mismo nombre
