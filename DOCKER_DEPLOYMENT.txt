📋 PASOS PARA USAR EN TU SERVIDOR

1️⃣  SETUP AUTOMÁTICO (Recomendado)
    Linux/Mac:  bash setup-docker.sh
    Windows:    setup-docker.bat

2️⃣  CONFIGURAR CREDENCIALES
    Editar .env con:
    - DB_HOST, DB_USER, DB_PASSWORD (PostgreSQL)
    - WHATSAPP_TOKEN, PHONE_NUMBER_ID, etc
    - BASE_URL=https://chat.irrigacionmalargue.net

3️⃣  MIGRAR DATOS (Si tienes MySQL)
    node migrate-db.js

4️⃣  DEPLOY
    docker build -t bot-irrigacion:prod .
    docker-compose up -d

5️⃣  VERIFICAR
    docker-compose ps
    curl https://chat.irrigacionmalargue.net/health
