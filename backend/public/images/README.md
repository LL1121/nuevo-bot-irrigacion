# Carpeta de Imágenes

Esta carpeta contiene las imágenes que el bot envía a los usuarios.

## Archivo requerido:

- **vencimientos.jpg** - Imagen con el calendario de vencimientos actualizado

### Instrucciones:

1. Guarda la imagen de vencimientos con el nombre exacto: `vencimientos.jpg`
2. El bot detectará automáticamente cuando la imagen esté disponible
3. Para actualizar la imagen, simplemente reemplaza el archivo con el mismo nombre
